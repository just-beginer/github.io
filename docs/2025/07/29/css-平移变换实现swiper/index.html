css 平移变换实现swiper

2025-07-29 | html/css

transform: translate(0, 0)的应用

<div class="container">
    <div class="slider">
        <div class="item item1"></div>
        <div class="item item2"></div>
    </div>
</div>

<style>
    .container {
        width: 100%;
        height: 100%
        overflow: hidden;
    }

    .slider {
        display: flex;
        width: 100%;
        height: 100%;
    }

    .item {
        flex-shrink: 0;
        width: 100%;
        height: 100%;
    }
    .item1 {
        background-color: gray;
    }
    .item2 {
        background-color: yellow;
    }
</style>

container容器是最外层, overflow:hidden当内容超出时隐藏
slider容器是用来滑动的父窗口, 利用flex而局, 多个子项目会依次排列，但只显示最左面的
item子项目flex-shrink设置为0, 表示不收缩, 宽度不会发生变化
最后利用transform: translate(0, 0)来平移实现轮播的效果, 并且加上transition过渡效果
